# led-matrix-display
A sleek enclosure and diffuser for the 8x32 WS2812B NeoPixel LED display

## Materials used:
- PETG-CF Black outer casing (any filament can be used)
- White PLA Pro filament for diffuser (any white or transparent filament can be used)
- Geekble Nano (ESP32-S3 but any Arduino/ESP32 can be used)
- M-F Jumper cables
- 8x32 WS2812B Matrix Display

## Assembly
Parts printed all on a Bambulab A1 mini
No glue, screws or additional hardware required for assembly.
Base can be assembled using a double dovetail joint although glue is optional.
Front diffuser panels can be slotted in before assembly.
USB-C Port for ESP32 powering display with 5V.
